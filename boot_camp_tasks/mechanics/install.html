Install Fusion 360

Standalone software (only Windows & Mac users)

  1. Go to https://manage.autodesk.com/products

  2. Log in when prompted.

  3. Select "Download" under "Fusion 360".

Fusion 360 in browser (better for users who are unable to install standalone program)

  1. Go to https://fusion.online.autodesk.com/ and follow the instructions and log in when prompted.
