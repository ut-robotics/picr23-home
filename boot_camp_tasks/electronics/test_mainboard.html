Test mainboard with a power supply and a motor

  1. Turn on the power supply.

  2. Set up power supply.

    1. Select one of the power supply channels.

    2. Set the voltage to 16 V. This is about the same voltage as the batteries.

    3. Set the current limit to 100 mA. Low current limit helps to prevent electronics from burning out from excessive power when something is shorted on the board. Current can be raised afterwards, if needed.

    4. Connect the leads to the selected power supply channel outputs.

    5. Do not turn on the output yet.

  3. Connect the leads to the mainboard.

  4. Turn on the output.

  5. Check that the current is not at the limit. If it is, then there might be something shorted on the board, or it might consume more current, in which case the current limit needs to be set higher. The higher current limit might not prevent components from burning out, so itâ€™s better to visually check for shorts. For example adjacent legs of a component might be bridged together with solder.

  6. Turn off the power supply outputs, if everything is fine with the mainboard.

  7. Connect motor driver boards to the mainboard.

  8. Connect motors to the motor driver boards.

  9. Turn on the power supply output.

  10. Check again that the current is not at the limit.

  11. Turn off the power supply outputs, if everything is fine.
